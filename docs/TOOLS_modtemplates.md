# Mod Example Templates

## [Surreal's Area Templates](https://github.com/SurrealDude/Mewgenics-Area-Template)

A INCREDIBLY helpful template of every single major part of a content mod.
Contains .gon and .fla formats for:
- Custom maps
- Custom NPCS
- Custom backgrounds
- Custom events
- Custom cutscenes
- Custom static objects
- Custom UI.
