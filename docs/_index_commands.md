# Commands

If debug console is turned on, you can open the console while the game is activated and playing at any moment by pressing the "`" key on the English keyboard.

There are a variety of commands that are available at any given moment in the game. Some commands are only usuable in certain scenarios (in battle, in event, in house, in cutscene, etc.).

The commands available are well-documented in the [Mewgenics Wiki.](https://mewgenics.wiki.gg/wiki/Debug_Console)
